# 🎬 Pixar-Lite Studio Pipeline

A lightweight studio-style 3D animation pipeline powered by Blender + GitHub Actions.

## 🚀 Features
- Automated rendering
- Shot-based workflow
- Dailies generation
- CI/CD animation pipeline

## 📂 Structure
- assets/ → reusable assets
- shots/ → animation shots
- pipeline/ → render scripts
- renders/ → output
- dailies/ → review videos

## ▶️ Usage
1. Add Blender file to:
   shots/SH001/lighting.blend

2. Push to GitHub

3. Go to Actions tab → rendering starts automatically
